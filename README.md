## 한국소프트웨어인재개발원 2차 프로젝트 [Gasan University DB]


## 프로젝트 목표

* Java와 DB, JDBC를 활용하여, 가상의 대학교 (가산대학교)의 종합정보시스템을 구현 한다.



## 사용한 기술 & 개발 환경

* JAVA 1.8
* Oracle Database
